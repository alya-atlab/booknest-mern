# BookNest

Full-stack MERN e-commerce platform for books with role-based access (user, author, admin).

## Overview

BookNest is a project that demonstrates a complete e-commerce workflow built with the MERN stack.  
The system supports multiple user roles, authentication, book management, and order processing.

## Features

- JWT authentication
- Role-based access control (user / author / admin)
- Books catalog browsing
- Shopping cart system
- Order checkout & history
- Author book management
- Admin dashboard

## Tech Stack

- MongoDB
- Express.js
- React + TypeScript (Vite)
- Node.js
- Mongoose
- JWT Authentication

## Project Structure

```
client/   → React frontend
server/   → Express + MongoDB backend
```

## Setup

### Backend

```bash
cd server
npm install
npm run dev
```


### Frontend

```bash
cd client
npm install
npm run dev
```
